Hugo主题切换为HBS

2024年5月12日2 分钟阅读 Yibel 博客搭建Hugo

目录

Hugo Bootstrap Theme(HBS) 是一个快速、响应式、多用途和功能丰富的 Hugo 主题。其通常用于博客和文档站点，地址：hugo-theme-bootstrap。因为此主题比较符合个人需求且文档较详细，所以将Hugo主题进行更改。

通过Git Submodule 安装主题

参考： 官方文档

安装到站点

先删掉之前的配置文档 hugo.yaml 或 config.yaml，然后进入 /blog 右键打开 Git Bash 执行以下命令

$ git submodule add https://github.com/razonyang/hugo-theme-bootstrap themes/hugo-theme-bootstrap
$ git clone https://github.com/razonyang/hugo-theme-bootstrap-skeleton /tmp/hbs-skeleton
$ mkdir config
$ cp -a /tmp/hbs-skeleton/config/* ./config
$ cp -r /tmp/hbs-skeleton/content/* ./content
$ cp -r /tmp/hbs-skeleton/archetypes/* ./archetypes
$ cp -r /tmp/hbs-skeleton/static/* ./static
$ cp -r /tmp/hbs-skeleton/assets/* ./assets
$ sed -i "s/theme:.*/theme: hugo-theme-bootstrap/g" config/_default/config.yaml
$ hugo mod npm pack
$ npm install
$ hugo server

配置

多语言（中英）

参考：配置多语言

  1. 编写配置文件 /config/_default/language.yaml
zh-hans:
  languageName: 简体中文
  weight: 1
  params:
    description: 
    keywords: 
    archive:
      dateFormat: 01-02
en:
  languageName: English
  weight: 2
  1. 在 /config/_default/config.yaml 中配置默认语言
defaultContentLanguage: zh-hans #放在前面才生效，我放在了theme下一行的位置
hasCJKLanguage: true
  1. 修改文件名，对于 /content 下的 index.md 文件，修改为 index.en.md，而后缀为 .zh-hans.md 的文件，删去 .zh-hans，这样语言切换页面才能生效

站点配置

参考 站点配置 进行配置即可

页面参数

参考 页面参数 ，建议在 archetypes/default.md 中设置好作为模板

网站logo和名称修改

在 config/_default/params.yaml 中进行如下设置（logo图片放在 blog/static/images）:

params.toml

brand = 'My Blog'
logo = 'images/logo.webp logo.webp.'

params.yaml

brand: My Blog
logo: images/logo.webp logo.webp.

params.json

{
   "brand": "My Blog",
   "logo": "images/logo.webp logo.webp."
}

此配置文件其他设置参考：站点参数
